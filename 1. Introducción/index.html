Cascading Style Sheet

Para poner un estilo se puede poner de tres formas.

-Como atributo style="color:blue;text-decoration:underline", pero no es recomendable.
-Como etiqueta style 'selector'{} en el head, para ver que tal va quedando.
-Como archivo externo, que lo haremos al final, para poder utilizarlo en varias páginas.

Ponemos p (selector que dice, a todas las p) y abrimos llave, color: tacatá;text-decoration: tucutú y cerramos llave (la última ; no hace falta)

Le asignamos una id a algo para después sólo cambiar eso
